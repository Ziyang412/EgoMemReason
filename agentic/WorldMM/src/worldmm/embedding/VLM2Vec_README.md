# VLM2Vec (vendored embedding backbone) — not shipped

WorldMM uses VLM2Vec as the visual embedding backbone. We do not vendor it here.
Install from upstream: https://github.com/TIGER-AI-Lab/VLM2Vec

Then either:
- pip install the package and import normally, or
- clone it into this directory: `git clone https://github.com/TIGER-AI-Lab/VLM2Vec.git`
